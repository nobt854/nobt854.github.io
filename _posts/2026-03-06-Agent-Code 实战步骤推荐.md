# 实战步骤

### 学习上手

先根据文档从简单到复杂，了解概念

### 实操目录

列下几个目录，从实战过程中持续的总结和记录

- **待处理**：任意从头脑中闪过的疑问，或者任意想通过 agent 来实现的实际需求，所有待处理、待验证来记录和驱动自己下一步如何实战

- **文档**：实战过程中收集到各类官方或技巧类的文档、链接，用于学习和回溯

- **常用**：实战过程需要频繁使用的，可能是操作提示词、流程步骤

- **总结**：经过实战总结出来能解决问题或者提高效率的内容，包括但不限于工具、流程、细节等
  - 工具、流程、细节等
  - agent code 的概念类，比如 SKILLS/MCP
  - 常用命令

- **尝试列表**：从上手到实战过程中所有通过 agent code 尝试过的场景，简单描述并记录对应的结果是怎样的，用于未来回溯

### 实战

基于一定了解后通过 agent code 重构实际工作流，尝试提高效率

### 持续

- **持续完善**：多多的实战并持续的完善以上目录

- **持续配置**：根据实战、总结、别人的好的分享，持续的完善 全局~/.claude/目录的配置、SKILLS、MCP 等

- **持续学习**：关注获取较新的 agent code 领域的最新功能、最新分享，对标自己上面列表总结和复盘、新的尝试待处理等
