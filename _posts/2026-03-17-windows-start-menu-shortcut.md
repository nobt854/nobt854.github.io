有个事儿可能很多人不知道：Windows 的开始菜单其实是可以手动"注入"快捷方式的。

---

## 🎯 核心目录

Windows 有一个特殊目录，放在这里的快捷方式会自动出现在开始菜单中：

```
C:\ProgramData\Microsoft\Windows\Start Menu\Programs
```

---

## 💡 使用场景

| 场景 | 说明 |
|------|------|
| **免安装软件** | 便携版应用不会自动创建开始菜单项 |
| **自定义脚本** | 批处理、PowerShell 脚本 |
| **命令行工具** | 想通过开始菜单快速启动 |
| **绿色软件** | 解压即用的程序 |

---

## 🔧 操作步骤

```
1. 找到目标程序的可执行文件
         ↓
2. 右键 → 创建快捷方式
         ↓
3. 将快捷方式移动到上述目录
         ↓
4. 打开开始菜单 → 所有应用 即可找到
```

---

## 📁 用户专属目录

如果只想对当前用户生效（不影响其他用户），可以使用：

```
%APPDATA%\Microsoft\Windows\Start Menu\Programs
```

---

## 🌟 实际应用

这个技巧的实用价值：

> **因为最好的系统，不是不能再加东西，而是不能再减东西。**

Windows 的设计保留了这种"可干预性"，让用户可以按自己的意愿组织开始菜单。

**常见用法**：
- 把常用的便携版工具集中管理
- 给复杂路径的程序创建简短快捷方式
- 为脚本设置专属图标和名称
- 快速访问深层目录的工具

---

## 📝 小结

一个简单但实用的小技巧，让你对开始菜单有完全的控制权。

下次下载了免安装软件，别再让它在下载文件夹里吃灰了，给它一个开始菜单的"户口"吧。
